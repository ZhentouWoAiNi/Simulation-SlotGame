Slot Game Simulation

*Description*
---
This is my first project, it is a slot game simulation intended for educational purposes.
To teach people that slot games like this can be controlled according to the wishes of the game owner.

*Feature*
---
the features are simple, there is only a page for top up, and simple elements

*How To Run*
---
How to run it is very easy, just type "flutter run" in the Visual Studio Code terminal, and just choose to use a browser such as (Chrome, Edge, Firefox, etc.)
